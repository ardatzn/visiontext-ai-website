# TZNtech | VisionText AI Static Website

VisionText AI is a static school project website for an e-commerce style AI product concept by TZNtech. The idea is an image-to-text system that converts multiple images into editable text for users such as online sellers, resellers, and small teams.

## Project Structure

- `index.html` - Home page
- `product.html` - Product overview
- `how-it-works.html` - Workflow explanation
- `pricing.html` - Demo pricing page
- `instagram.html` - Instagram marketing plan
- `about.html` - About the school project
- `contact.html` - Contact form design
- `assets/css/style.css` - Shared styles for all pages
- `assets/js/main.js` - Shared JavaScript for navigation and demo interactions
- `assets/img/` - Image assets used by the pages

## Features

- Responsive multi-page layout
- Shared navigation and footer on every page
- Home hero section and feature cards
- Fake multi-image upload demo
- Example text extraction result area
- Pricing cards
- Instagram marketing strategy page
- Contact form design

## How To Open

1. Open the project folder.
2. Double-click `index.html` in a browser, or open it with your browser manually.
3. Navigate between pages using the menu links.

## Notes

- This project uses only HTML, CSS, and JavaScript.
- There is no backend, database, or package manager.
- The upload demo and contact form are front-end only and do not send real data.
